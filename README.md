## INSTALLATION GUIDE
1. Mongoose

2. Express

   ```npm i dotenv```
4. Cors
   ```npm i cors``` 
5. Json Web Token
   ```npm i jsonwebtoken```
6. Bcrypt
   ```npm i bcrypt``` 
7. Nodemon 
   ```npm i nodemon```
